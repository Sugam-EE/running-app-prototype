# Running App Prototype (Nepal)

## Overview
This project uses **MATLAB Mobile** sensor data to analyze running gait and track performance. 
As an electronics engineering student and aspiring athlete, my goal is to build a tool 
that works specifically well in high-altitude environments like Kathmandu.

## Current Goals
* [ ] Capture raw Accelerometer data from my phone.
* [ ] Develop a Signal Processing filter to remove "noise" from steps.
* [ ] Create a "Peak Detection" algorithm to count steps accurately.
* [ ] Integrate GPS data to track distance and altitude gain.

## Hardware/Software
* **Device:** Smartphone (Android/iOS) via MATLAB Mobile
* **Processing:** MATLAB R2025a
* **Focus:** Digital Signal Processing (DSP) & Biomechanics
